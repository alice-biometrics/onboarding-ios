Alice Onboarding iOS component allows the automatic capture of documents and video selfie of the user in real time from the camera of your device. 
The main features are:

- Automatic capture of documents and video selfie of the user in real time from the camera of your device.
- Communication with the onboarding API to facilitate rapid integration and development.
- Management of the onboarding flow configuration: requested documents and order.
